#  Superstore Sales Analysis Using SQL

##  Project Overview
This project analyzes retail sales data using SQL to identify business performance drivers, regional profitability trends, product performance, and the impact of discounts on profit.

##  Tools Used
MySQL Workbench
SQL
GitHub

##  Business Questions
Which region generates the highest profit?
Which category is most profitable?
Which sub-categories are loss-making?
How do discounts affect profitability?

##  SQL Techniques Used
Aggregations (SUM, AVG)
GROUP BY
HAVING
ORDER BY
Business performance analysis

##  Key Insights
Regional profitability varies significantly.
Some sub-categories generate losses despite strong sales.
Excessive discounts negatively impact profit.
High-performing categories contribute disproportionately to business success.

##  Business Recommendations
Optimize discount strategies.
Improve low-performing regions.
Focus on profitable categories.
Address loss-making sub-categories.
